5 bahasa pemograman populer

5 Bahasa Pemrograman Populer

by Fachrul Ahaddin | in Website Basic | View 656

Pasti pada tahap awal-awal belajar membuat website atau belajar programming, kita pasti bingung mau belajar apa ya ? programming apa sih yang paling menjanjikan untuk ke depannya ? karena memang programming itu banyak macamnya, banyak spesialisasinya. Nah kita yang baru mau belajar dihadapkan dengan banyaknya macam pilihan bahasa pemrograman. Tentunya dengan kelemahan dan kelebihan masing-masing. Selain itu faktor gaji juga sering menjadi pertimbangan dalam mengambil keputusan bahasa apa yang akan kita pelajari. Cukup masuk akal karena memang orientasi awal lebih banyak ke arah gaji. Disini saya mencoba membantu kalian semua untuk mereferensikan bahasa apa sih yang paling banya di bahas dan digunakan. Saya mengambil data dari survey sebuah website forum terbesar yaitu stackoverflow. Stackoverflow rutin mengeluarkan data setiap tahunnya. Teman-teman pasti pernah dengar atau bahkan sering menggunakan kan? Oke disini saya akan bahas 5 besar bahasa pemrograman, scripting dan bahasa markup yang sering di bahas di stackoverflow pada tahun 2020 kemarin.

Javascript

Javscript menempati urutan pertama. Karena javascipt mengalami perkembangan yang sangat pesat. Framework-framework yang memanjakan programmer dengan fitur-fitur yang memudahkan programmer menjadi salah satu daya tarik javascript. Jadi tidak heran javascript terus mengalami perkembangan dan mungkin akan terus berkembang sampai beberapa tahun ke depan.

HTML & CSS

Kalau ini mah gak usah ditanya lagi ya, karena memang basic nya dan javascript pun masih menggunakan jasa HTML & CSS dalam pemrosesannya

SQL

Bahasa SQL masih menjadi primadona setidaknya sampai tahun 2020 kemarin. Tentunya Karena sudah sangat familiar dengan bahasa-bahasanya, jadi banyak programmer yang masih menggunakannya. Ditambah lagi dukungan dari DBMSnya yang banyak membuat SQL masih sangat digandrungi para programmer.

PYTHON

Nah ini bukan jenis ular ya, tapi sebuah bahasa pemrograman. Phyton sudah banyak digunakan di luar Indonesia dan menjadi salah satu bahasa popular disana. Dukungan library yang sangat banyak menjadi salah satu alasan phyton sangat digemari. Selain itu phyton juga mendukung IoT, jadi bisa menghubungkan benda-benda di lingkungan sekitar kita ke dalam sebuah jaringan-jaringan yang terhubung satu sama lain.

Java

Java sampai saat ini masih cukup Berjaya dengan dukungan berbagai macam platform. Merupakan salah satu bahasa yang berorientasi objek, jadi cukup mudah dipelajari. Sampai detik ini java masih sangat digemari bagi programmer-programmer.

Oke itulah 5 besar hasil dari survey stackoverflow selama tahun 2020. Jika teman-teman mau membaca lebih lengkapnya, bisa klik disini , oke semoga bisa membantu untuk mereferensikan bahasa pemrograman ya, terima kasih.
